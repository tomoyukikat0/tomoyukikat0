# 職務経歴書

## 【経歴概要】

薬局・ドラッグストアでの薬剤師経験を経て、ヘルスケア系スタートアップの 1 人目の正社員エンジニアとして、自社サービスの新規構築および機能追加を行ってきました。
開発者としては主にバックエンドを担当し、実装だけではなく、企画の段階から入り、要件定義・デザイン・設計・実装・保守運用まで幅広く経験してきました。
また開発業務だけではなく、エンジニア採用や技術広報などの活動も行ってまいりました。

## 【スキル】

| 得意技術 | Ruby on Rails |
| 業務経験 | Ruby on Rails, React, Next.js, JavaScript, TypeScript, Python, Rust, MySQL, PostgreSQL, Docker, Terraform, Google Cloud, gRPC, LangChain, LangSmith, LangGraph, PromptLayer, Looker studio, Figma etc|
